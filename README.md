# 🎮 Pokédex de Jean-François

Une galerie interactive de Pokémons avec système de recherche et évolutions.

## 🚀 Comment ouvrir le projet

1. Téléchargez ou clonez ce dossier
2. Ouvrez le fichier `index.html` dans votre navigateur web
   - Double-cliquez sur `index.html`, ou
   - Faites un clic droit → "Ouvrir avec" → votre navigateur préféré

C'est tout ! 🎉

## 📁 Structure du projet

```
Pokedex/
├── index.html          # Page principale
├── style.css           # Styles CSS
└── pokepics/          # Images des Pokémons
```
